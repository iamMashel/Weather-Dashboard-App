# 🌤️ Weather Dashboard

A simple weather app using Vanilla JavaScript and the OpenWeatherMap API.

## 🚀 Features
- Real-time weather search
- Temperature, wind, and humidity display
- City search with error handling

## 📦 Technologies
- HTML5
- CSS3
- JavaScript (ES6)
- OpenWeatherMap API
